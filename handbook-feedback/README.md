# Handbook Feedback

This directory contains feedback, errata, and suggestions for the AI × Web3 School Handbook.

## How to submit feedback

Each feedback entry should be a markdown file following this naming convention:
`YYYY-MM-DD-brief-title.md`

## Required fields in each feedback entry

- **Handbook URL**: Link to the specific page/section
- **Issue type**: Typo / Unclear concept / Outdated info / Structure suggestion / Missing content
- **Description**: What the issue is
- **Suggestion**: Proposed fix or improvement
- **Source**: How I discovered this (reading, experiment, discussion)

## Purpose

This feedback is publicly visible and can be indexed. The Handbook maintainers (LXDAO & ETHPanda) use community feedback to continuously improve the content.
