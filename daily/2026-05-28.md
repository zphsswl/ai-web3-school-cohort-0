## 💡 Core Takeaway

**Agent Workflow = 把概率模型放进确定性流程。**

一个成熟的链上 Agent 不能只有 prompt 和 "下一步推理"。它需要：
- **Task Graph** — 把目标拆成有依赖的节点
- **State Machine** — 知道当前在哪一步，失败后如何恢复
- **Human-in-the-loop** — 人在关键风险点介入，不确认低风险步骤
- **Retry/Fallback** — 不同错误不同策略，不盲目重试
- **Trace** — 每一步可复盘，区分模型错/工具错/policy 漏/人错
- **Regression Set** — 改模型或工具前跑固定用例，确认没有从"拒绝危险"退化成"更顺但更危险"

> *Without workflow, projects quickly become "model directly calls tools" — fast in demos, insufficient with real assets.*
